## üö® CRITICAL: OpenClaw Security Issues

**Source:** The Verge  
**URL:** https://www.theverge.com/news/874011/openclaw-ai-skill-clawhub-extensions-security-nightmare

**What happened:**
- 1Password discovered malware in hundreds of ClawHub skills
- 414 malicious skills uploaded between Jan 27 - Feb 2
- Malware disguised as crypto trading tools, steals API keys, wallet private keys, SSH credentials, browser passwords
- Even most-downloaded add-ons contained malware delivery mechanisms
- Skills use markdown files that can contain malicious instructions

**Our response:**
- Peter Steinberger (OpenClaw creator) is addressing it
- New requirements: GitHub account must be 1+ week old to publish skills
- Reporting mechanism added
- Still risk of malware sneaking through

**Action items:**
- Be extremely cautious when installing any skills from ClawHub
- Review existing installed skills
- Consider security implications for Dino's setup
