## Wie finden?

### LinkedIn Suche
```
Suchbegriffe:
- "Freelance Designer" + Deutschland
- "Freelance Entwickler"
- "Virtual Assistant" + DACH
- "Marketing Freelancer"
- "Selbstständig" + [Beruf]
```

### Filter nutzen
- Standort: Deutschland, Österreich, Schweiz (oder international EN)
- Connections: 2nd degree (können Anfrage senden)
- Aktivität: "Posted in last 30 days" (aktive User)

## Connection Request (WICHTIG: Kein Pitch!)

### Template 1: Gemeinsames Interesse
```
Hi [Name], 

sehe dass du auch im [Design/Marketing/etc.] Bereich unterwegs bist. 
Würde mich freuen, uns zu vernetzen!

Dino
```

### Template 2: Ihre Arbeit loben
```
Hi [Name],

bin gerade über dein Portfolio gestolpert — starke Arbeiten! 
Besonders [konkretes Beispiel] hat mir gefallen.

Freue mich auf den Austausch.

Dino
```

### Template 3: Neutral
```
Hi [Name],

vernetze mich gerade mit anderen [Freelancern/Selbstständigen] 
im [Bereich]. Würde mich freuen!

Beste Grüße,
Dino
```

## Follow-Up Nachricht (1-2 Tage nach Annahme)

### Template A: Direkt mit Value
```
Hey [Name], danke fürs Vernetzen!

Kurze Frage: Wie viel Zeit verbringst du aktuell mit 
wiederkehrenden Aufgaben — Emails, Recherche, Reporting?

Ich helfe Freelancern, solche Tasks mit AI zu automatisieren. 
Die meisten sparen 5-10h pro Woche.

Falls interessant: Ich biete einen kostenlosen 15-Min Check an, 
wo ich dir zeige was bei dir möglich wäre. Kein Sales-Pitch, 
nur konkrete Tipps.

Wäre das was für dich?
```

### Template B: Sanfter Einstieg
```
Hey [Name], freut mich!

Was treibt dich gerade so um? Arbeitest du aktuell 
an spannenden Projekten?

(Ich bin im Bereich AI-Automatisierung unterwegs und 
helfe Freelancern, mehr Zeit für die wichtigen Sachen zu haben.)
```

### Template C: Problem-fokussiert
```
Hey [Name],

eine Sache die ich oft von [Designern/Textern/etc.] höre: 
Die eigentliche Arbeit macht Spaß, aber der ganze 
Admin-Kram drumherum frisst Zeit.

Geht dir das auch so? Falls ja — ich zeige Freelancern 
in 15 Min, welche Tasks sie mit AI automatisieren können.

Interesse?
```

## Wenn sie antworten

### "Ja, interessiert mich"
```
Super! Hier ist mein Calendly-Link: [LINK]

Such dir einen Slot aus der passt. Ich zeig dir dann 
konkret was bei dir möglich ist.

Bis bald!
```

### "Was genau machst du?"
```
Ich analysiere Workflows von Freelancern und zeige, 
welche Aufgaben sich mit AI-Tools automatisieren lassen.

Zum Beispiel:
- Email-Vorlagen die sich selbst schreiben
- Recherche die in Minuten statt Stunden fertig ist
- Content-Erstellung mit AI-Unterstützung
- Automatische Reportings

Im 15-Min Call gucken wir uns an, was bei DIR am meisten 
Sinn macht. Kostenlos, kein Verkaufsgespräch.

Hast du Zeit diese Woche?
```

### "Klingt interessant, aber gerade keine Zeit"
```
Verstehe! Kein Stress.

Soll ich mich in 2-3 Wochen nochmal melden? 
Oder schreib mir einfach wenn es besser passt.
```

### Keine Antwort nach 3-5 Tagen
```
Hey [Name], kurzes Follow-up — 

hast du meine letzte Nachricht gesehen? 
Falls das Thema AI-Automatisierung gerade nicht relevant ist, 
kein Problem! Wollte nur sichergehen dass es nicht untergegangen ist.
```
