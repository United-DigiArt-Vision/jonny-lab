# X Thread Posting — Anleitung

## Methode 1: Script (bevorzugt, Token-sparend)
**Script:** `tools/post-thread-instructions.md`
**Ablauf:** Thread-Datei → 1 JS-Evaluate → Post all → Fertig
**Kosten:** ~3 Browser-Calls statt 30+

## Methode 2: Manuell per Browser (Fallback)
**Ablauf:** Für jeden Tweet: Add post → Snapshot → Type → Snapshot
**Kosten:** ~$5 pro Thread (viele Snapshots)
**Vorteil:** Funktioniert zuverlässig, gute Ergebnisse bestätigt
**Wann nutzen:** Wenn Script-Methode nicht funktioniert
